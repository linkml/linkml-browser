DisMech Review Help

Guidance for reviewing disorder mechanism cards

Project overview

This project focuses on reviewing disease mechanisms, phenotypes, and related evidence in the DisMech dataset. Use search and facets to narrow to a set of disorders, then evaluate the key claims on each card.

Card layout

  • Disorder and ID identify the record.
  • Mechanisms, phenotypes, genes, treatments, and related evidence appear as fields.
  • Source links open the underlying data file when available.

Evaluation fields

  • Mechanism Confidence (1-5): overall confidence in the mechanism description.
  • Evidence Strength (low/medium/high): how strong the evidence appears.
  • Needs Follow-up (yes/no): flag items that require deeper review.
  • Curation Notes: free text notes or corrections.

Status

Status is tracked automatically:

  • pending no evaluation yet
  • draft some evaluation data entered
  • submitted explicitly submitted
  • discarded explicitly discarded

Saving evaluations

Desktop app: choose “Save Evaluations As…” once to set the autosave file. Changes write to that file automatically.

Browser: evaluations are stored in localStorage. Use Export Evaluations to download a JSON file.
