Sentiment Analysis Review Help

Guide for evaluating sentiment model predictions

Project overview

This project focuses on human evaluation of sentiment predictions. Each card shows the original text and the model’s predicted sentiment label with a confidence score.

How to review

  • Read the text and the predicted sentiment label.
  • Use the thumbs-up/down on the card to mark whether the prediction is correct.
  • Add a note if the example is nuanced or requires follow-up.

Evaluation fields

  • Thumbs up/down: quick correctness signal.
  • Reviewer Notes: short rationale or corrections.

Status

  • pending no evaluation yet
  • draft some evaluation data entered
  • submitted explicitly submitted
  • discarded explicitly discarded

Saving evaluations

Desktop app: choose “Save Evaluations As…” once to set the autosave file. Changes write to that file automatically.

Browser: evaluations are stored in localStorage. Use Export Evaluations to download a JSON file.
